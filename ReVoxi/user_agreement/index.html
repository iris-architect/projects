Политика конфиденциальности приложения "ReVoxi" (Версия 1.0)

Дата вступления в силу: 10.05.2026

Настоящая Политика конфиденциальности описывает, какие данные обрабатывает приложение ReVoxi.

1. Какие данные собирает приложение

Приложение не собирает, не хранит и не передает персональные данные пользователей.

Приложение не требует регистрации и не использует учетные записи пользователей.

2. Использование микрофона

Приложение использует доступ к микрофону устройства исключительно для передачи звука в режиме реального времени из микрофона в наушники пользователя.

Звук:

  • не записывается
  • не сохраняется
  • не анализируется
  • не передается третьим лицам
  • не отправляется на сервер

Обработка звука происходит только локально на устройстве пользователя.

3. Передача данных третьим лицам

Приложение не передает данные пользователей третьим лицам.

4. Хранение данных

Приложение не хранит пользовательские данные.

5. Безопасность

Поскольку приложение не собирает и не хранит данные пользователей, риск утечки персональных данных отсутствует.

6. Изменения политики конфиденциальности

Разработчик может обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна в приложении.
