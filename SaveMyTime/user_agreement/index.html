Аккаунт — уникальная учетная запись Пользователя в Приложении.

Анонимный Аккаунт — временный Аккаунт, создаваемый без указания адреса электронной почты и пароля.

Зарегистрированный Аккаунт — постоянный Аккаунт, создаваемый с использованием адреса электронной почты и пароля.

Контент Пользователя — данные, вводимые Пользователем в Приложение.

Сервис хранения данных — облачная платформа Firebase (Google LLC), используемая для хранения данных Аккаунта.

  • Расписание.
  • Имя, возраст, хронотип и уровень концентрации пользователя.

Все данные (кроме локальных настроек) хранятся в Firebase (Google LLC). Данные могут физически размещаться на серверах, расположенных за пределами страны проживания Пользователя.

Пользователь соглашается с политикой конфиденциальности Google.

Данные используются для обеспечения функционала Приложения, синхронизации данных, восстановления Аккаунта.

Данные не используются для рекламы.

Передача возможна только:

  • Инфраструктурным поставщикам (Firebase).
  • Если это требуется законом.

Приложение не продает пользовательские данные.

Пользователь может удалить Аккаунт в любой момент. После удаления:
